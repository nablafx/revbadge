\begin{abstract}
This document serves as both the official user manual and a compilation demonstration for the \texttt{imeantit} package. We present examples of how to address different reviewers using distinct color schemes without breaking tough, two-column academic structures.
\end{abstract}

\section{Introduction \& Package Purpose}
With the increasing rigor of peer-review processes, keeping track of manuscript modifications has become crucial. The \texttt{imeantit} package provides a dual-component architecture consisting of the multi-line \texttt{imeant} block environment and the \texttt{\textbackslash imeantinline} macro for tight mid-sentence updates.

For example, a minor structural adjustment can be highlighted as an \imeantinline[colframe=gold, colback=purpb]{1}{rev:short}{structural block} modification, which automatically links to your response letters via \texttt{cleveref}.
